Detailed Description

This class is an abstract implementation of PepperExporter and cannot be instantiated directly. To implement an exporter for Pepper, the easiest way is to derive this class. For further information, read the javadoc of PepperManipulator and the documentation of u.hu-berlin.de/saltnpepper.

See also
PepperManipulator
