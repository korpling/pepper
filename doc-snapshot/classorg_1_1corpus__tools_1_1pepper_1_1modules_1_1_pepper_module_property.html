Detailed Description

The class PepperModuleProperty offers a possibility to describe a property, which is used by a specific PepperModule.

A list of PepperModuleProperty objects is requested by Pepper to give an overview of properties to the user. T specifies the type of this property. This class contains:

  1. the name of this property
  2. a description of its sense to present it to the user
  3. a field determining if this property is required by the corresponding PepperModule object
